# 🔐 User Authentication System

## 📌 Overview
This project is a simple User Authentication System that allows users to register and log in securely. It demonstrates the basic workflow of authentication used in modern web applications.

---

## 🚀 Features
- User Registration
- User Login
- Form Validation
- Password Security (Hashing)
- Session Management
- Logout Functionality

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: (PHP / Node.js / Python)
- Database: MySQL / MongoDB

## ⚙️ Installation

### 1. Clone Repository
```
git clone https://github.com/your-username/user-auth-system.git
cd user-auth-system
```

### 2. Setup Database
- Create a database (user_auth)
- Import schema.sql file

### 3. Configure Database
Update your database credentials:
```
Host: localhost
Username: root
Password: your_password
Database: user_auth
```

### 4. Run Project
- For PHP: Use XAMPP/WAMP and place project in htdocs
- For Node.js:
```
npm install
npm start
```
